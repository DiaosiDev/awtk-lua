# awtk-lua

awtk lua bindings.

## 准备

1.获取awtk并编译

```
git clone https://github.com/zlgopen/awtk.git
cd awtk; scons; cd -
```

2.获取awtk-lua并编译

```
git clone https://github.com/zlgopen/awtk-lua.git
cd awtk-lua
scons
```

## 更新绑定

```
./sync.sh
```

> 在非bash终端(如Windows平台的cmd.exe)，需要根据sync.sh的内容手工执行相应的命令。

## 运行

```
./bin/awtkRun demos/xxxx.lua
```
> 请把xxxx.lua换成具体的lua文件。
